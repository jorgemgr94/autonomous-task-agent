# Autonomous Task Agent

Autonomous AI agent capable of:
- Analyzing open-ended tasks
- Making decisions
- Executing external tools via APIs
- Maintaining state and traceability

## Architecture Overview
This project follows an API-first backend architecture using FastAPI.
The agent is designed as a reasoning loop with controlled tool execution.

## Goals
- Demonstrate autonomous decision-making
- Safe and structured tool usage
- Production-oriented architecture
